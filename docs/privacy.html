달그미데스크 개인정보 처리방침

최종 개정: 2026-04-24 · 배포 주체: 넥스트랩(NextLab)

1. 개요

달그미데스크(이하 "확장")는 팀 내부에서 사용하는 개인 북마크 시작페이지 Chrome 확장입니다. 확장은 사용자가 입력한 북마크·섹션 데이터를 Supabase 서버에 저장하며, 그 외의 개인정보는 수집하지 않습니다.

2. 수집하는 정보

  • 이메일 주소: 로그인 식별자. Supabase Auth 가 관리.
  • 북마크·섹션 데이터: 사용자가 직접 입력한 제목·URL·카테고리명.
  • 세션 토큰: 자동 로그인을 위해 브라우저 localStorage 와 IndexedDB 에 저장.

확장은 다음을 수집·저장·전송하지 않습니다:

  • 방문 기록, 탭 내용, 키 입력
  • 위치 정보, 연락처, 결제 정보
  • 분석·광고용 추적 데이터

3. 데이터 저장 위치

  • 서버: Supabase (지역 ap-northeast-2, 서울) PostgreSQL. 행 수준 보안(Row-Level Security) 정책으로 본인 데이터만 접근 가능.
  • 로컬: 브라우저 IndexedDB 오프라인 캐시 (본인 PC 내부).

4. 제3자 공유

확장은 사용자 데이터를 어떤 제3자와도 공유하지 않습니다. 단, Supabase 가 데이터 저장 인프라로서 관여합니다. Supabase 의 개인정보 처리방침은 https://supabase.com/privacy 에서 확인 가능합니다.
